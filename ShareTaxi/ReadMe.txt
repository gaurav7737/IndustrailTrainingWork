To run the program -->>  run the TaxiStand.java file

Two threads are running-
1. Generate Passenger
2. Generate Taxi

Notes:
1. New passenger is generated in every 3 seconds with a random destination.
2. New taxi arrives in every 50 seconds. 
3. Capacity of taxi stand = 5 taxi
4. If no taxi is available for passenger then it is added in Passenger Queue.
